# FL01 - TMA Bands Channel

## Overview
**FL01** is a dynamic Triangular Moving Average (TMA) Bands indicator. It envelopes the price action with upper, lower, and median bands, creating a smooth, dynamic price channel that acts as a robust measure of support and resistance. It is exceptionally good at helping traders identify ranging markets, overextended trends, and potential mean-reversion zones.

## Key Features
- **TMA Channel Logic:** Smooth and dynamic boundaries that act as adaptive support and resistance levels.
- **Multi-Timeframe (MTF) Capabilities:** Project the TMA bands of a higher timeframe (e.g., H4) seamlessly onto your current chart (e.g., M15).
- **Interpolation Engine:** Smoothly interpolates higher timeframe values on lower timeframe charts to prevent visual "blockiness" and ensure a clean look.
- **Comprehensive Alerts System:** Choose to receive alerts via Popup, Email, or Sound when the price penetrates the upper or lower bands.

## How to Install
1. Download the compiled files (`.ex4` for MT4 or `.ex5` for MT5) from this repository.
2. Open your MetaTrader platform.
3. Click on `File` -> `Open Data Folder` in the top menu.
4. For MT4, navigate to the `MQL4/Indicators` folder. For MT5, navigate to the `MQL5/Indicators` folder.
5. Paste the downloaded files into the folder.
6. Refresh the "Navigator" panel in MetaTrader, or simply restart the platform.
7. Find the indicator in the Navigator, drag and drop it onto your desired chart, and start trading!
